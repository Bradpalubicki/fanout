## Their Features (for parity checklist)
- [ ] Send posts (text, images, video, scheduled or immediate)
- [ ] Delete posts via post ID
- [ ] Post history with status + metadata
- [ ] Comment management (retrieve, post, moderate)
- [ ] Review management (Facebook, Google Business Profile)
- [ ] Auto hashtags (keyword-based, real-time popularity)
- [ ] DM / messaging (text, images, videos)
- [ ] Conversation history
- [ ] Automated responses
- [ ] Webhooks for interactions + completed posts
- [ ] Analytics (likes, retweets, clicks, impressions)
- [ ] Public data access (brand monitoring, competitive)
- [ ] FB Ads / Boosted Posts
- [ ] 3rd party: Notion, Airtable, Make, Bubble
